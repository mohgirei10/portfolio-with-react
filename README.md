## 🚀 Technical Highlights

- **React SPA:** Built with a component-based architecture for speed and scalability.
- **Responsive Design:** Mobile-first approach using W3.CSS and custom media queries.
- **Dynamic Theming:** Custom Dark/Light mode toggle with **Persistent State** (via LocalStorage).
- **Automated Deployment:** Integrated CI/CD pipeline using **GitHub Actions**.
- **Contact Integration:** Functional backend-less contact form powered by **Formspree**.
